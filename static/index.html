Getting Started

  1. Select your trainer — choose which training tool you used.
  2. Browse to your workspace — select the run/workspace folder.
  3. Pick a training run — the dataset is auto-detected from the config.
  4. Choose comparison mode — Person's Likeness or Style Similarity.
  5. Select steps — choose all or narrow to a range.
  6. Run — images are compared and ranked by similarity.

Powered by InsightFace (ArcFace) for face recognition and CSD for style comparison.
