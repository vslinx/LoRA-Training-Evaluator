Getting Started

  1. Select your trainer — choose which training tool you used.
  2. Browse to your workspace — select the run/workspace folder.
  3. Pick a training run — the dataset is auto-detected from the config.
  4. Select steps — choose all or narrow to a range.
  5. Run — faces are detected, compared, and ranked by likeness.

Powered by InsightFace (ArcFace) for face detection and recognition.
